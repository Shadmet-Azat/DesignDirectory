# The movement vocabulary — core 13

Pick the **one** movement that fits the subject's intent (fittest, not flashiest), commit, and carry its
signature through. Each entry: *what it is · when it fits · the cliché-trap · the signature execution · pairs-with
(→ design-techniques) · currency.* These are **targets to exceed**, not templates to copy.

## MACRO-SIGNAL (2026) — apply to every choice
The field is reacting *against* sterile, automated, "AI-polished" aesthetics — toward **warmth, texture, the
handcrafted, the human, and restraint.** Two consequences: (1) when an axis is free, spend it on warmth/tactility,
not a louder default; (2) **typography is center stage** (storytelling, oversized, custom), **maximalism is
rising**, and **minimalism stays but warms**. Trends move — re-check before treating any currency note as fixed.

---

## 1 · Swiss / International typographic
- *What:* grid-disciplined, type-led, objective; asymmetric grid, flush-left, generous space, photography > illustration.
- *Fits:* editorial, institutions, design-forward brands, portfolios — authority + clarity.
- *Cliché-trap:* "Helvetica + grid = done" sterile corporate; OR calcified big-type-on-stark-white + whitespace that now reads as **default-tasteful**. Emptiness mistaken for rigor.
- *Signature:* a real type **system** (modular scale, 2–3 weights, baseline grid), asymmetric *tension* not centered, one bold scale jump, mono + 1 accent, full-bleed photography, the grid visible. **Warm it** — off-white, a characterful grotesque, not Helvetica.
- *Pairs-with:* split-text, variable-font, marquee, restrained reveal-on-scroll.
- *Currency:* type-led is huge in 2026, but plain swiss-on-white is the default — needs tension + warmth to clear the floor.

## 2 · Editorial-magazine
- *What:* print-magazine logic on web — drop caps, pull quotes, columns, run-in heads, art-directed spreads, captions/bylines.
- *Fits:* content/story-led sites, brands with a POV, long-form, fashion/culture.
- *Cliché-trap:* the broadsheet **hairline-rules + dense columns + zero-radius** default; generic blog-with-a-serif; numbering as decoration not sequence.
- *Signature:* real art direction **per section** (not one template), pull quotes as type events, drop caps, asymmetric image/text spreads, a display+body pairing, deliberate measure. Numbering only if a real sequence.
- *Pairs-with:* split-text, reveal-on-scroll, clip-wipe, scrollytelling.
- *Currency:* strong (type-as-storytelling).

## 3 · Neobrutalism
- *What:* raw, high-contrast, thick borders, hard offset shadows, clashing saturated color, system/mono type, exposed structure.
- *Fits:* dev tools, indie/creative brands, Gen-Z products — "anti-corporate / honest."
- *Cliché-trap:* the **template kit** (yellow + black border + hard shadow on everything) — now its own default. Raw-for-raw's-sake is **past peak**.
- *Signature:* **"refined neo-brutalism"** — ONE brutalist gesture (an oversized head, a hard-bordered key module), disciplined elsewhere; a *characterful* grotesque not just default mono; *intentional* clashing palette; structure exposed but legible. Refined > raw.
- *Pairs-with:* hard stateful hover, marquee, sticky, drag.
- *Currency:* maturing/refining; past the raw peak.

## 4 · Minimal-refined (dark-premium + warm minimal)
- *What:* restraint as luxury; few elements, precise spacing, one accent. Two modes — dark-premium (near-black + 1 accent) and the **2026 warm minimal** (off-white, soft gradient, tactile).
- *Fits:* premium SaaS, hardware, luxury-adjacent — calm = trust.
- *Cliché-trap:* the **near-black + single acid-green/vermilion** default; OR **cold sterile minimal** (what 2026 reacts against); emptiness as minimalism.
- *Signature:* obsessive spacing/type detail, ONE confident accent, micro-interaction craft, depth via subtle layering. **Warm it** — soft gradient, off-white, tactile grain, human copy; minimal + warmth + emotional engagement, not cold.
- *Pairs-with:* stateful hover, grain, subtle reveal, restrained mesh gradient.
- *Currency:* minimal stays, but the live version is **warm, not sterile**.

## 5 · Maximalism
- *What:* density, layering, bold clashing color, multiple type families, ornament — "more is more."
- *Fits:* culture, music, fashion, events — energy/personality.
- *Cliché-trap:* chaos with **no hierarchy** (noise); "chaos packaging" with no organizing logic; maximal = cluttered/unusable.
- *Signature:* **maximal surface over a rigorous structure** — an underlying grid disciplines the abundance, one focal hierarchy survives the density, color collisions are *intentional*, layering has depth. "Dual aesthetics" — switch min/max by context.
- *Pairs-with:* marquee, kinetic type, blend modes, layered depth, custom cursor.
- *Currency:* **rising.** Discipline = structure under the chaos.

## 6 · Retro-futurism (primary) / Y2K (adjacent)
- *What:* past visions of the future — 70s/80s sci-fi, chrome, CRT/scanlines, neon-on-dark, retro gradients, sci-fi type. Y2K = early-2000s chrome/bubble/saturation.
- *Fits:* tech/AI brands wanting nostalgia + future, gaming, music, bold consumer.
- *Cliché-trap:* the **sticker-pack** version (chrome gradient + scanline = "retro"); vaporwave-by-numbers; nostalgia with no concept.
- *Signature:* commit to a **specific era's** language (70s sci-fi ≠ Y2K chrome ≠ 80s Memphis); period-accurate type + palette; carry the future-nostalgia *tension*. Saturation revival = bold color is back.
- *Pairs-with:* scramble/decode, grain/CRT, mesh gradient, glitch shader, kinetic type.
- *Currency:* **strongly rising 2026.**

## 7 · Skeuomorphism (revival — light)
- *What:* digital mimics physical — texture, depth, material cues. 2026 form is **light skeuo / neumorphism / frosted glass** (soft shadows, tactile), NOT heavy 2010 leather-stitch.
- *Fits:* products wanting warmth/tactility/familiarity; audio/creative tools; the anti-flat reaction.
- *Cliché-trap:* heavy 2010 skeuo (leather/felt/torn-paper) = dated; **pure neumorphism = low-contrast a11y failure**; texture as sticker.
- *Signature:* **light touch** — soft realistic shadows + subtle material texture + tactile depth, but legible/accessible; ONE material metaphor true to the subject; modern type so it's not retro. Apple **Liquid Glass** = the high-end nod.
- *Pairs-with:* glass, liquid glass, layered depth, grain, stateful hover.
- *Currency:* **genuinely reviving** — but **light**; watch the contrast/a11y trap.

## 8 · Organic / biomorphic
- *What:* natural forms — flowing curves, blobs, hand-drawn, soft shapes, earthy/botanical, asymmetric natural rhythm. "Cyber-organic" = organic + tech.
- *Fits:* wellness, beauty, food, sustainability, human/warm brands.
- *Cliché-trap:* the default **blob + soft gradient + rounded-sans wellness template**; organic as decoration not structure; the generic "aurora blob."
- *Signature:* organic logic in the **layout and motion** (flowing, asymmetric, natural easing), not just blob decals; **hand-drawn/illustrative assets**; earthy palette with intent; tactile texture (grain/paper). Tactile craft, naive/hand-drawn, anti-sterile.
- *Pairs-with:* SVG morph, blob cursor, grain, soft reveal, mesh gradient.
- *Currency:* **strong 2026** (organic flow, tactile, naive).

## 9 · Luxury / fashion-editorial
- *What:* restraint + craftsmanship + sensory richness; refined type, generous space, art-directed full-bleed imagery/video, slow elegant motion.
- *Fits:* luxury goods, fashion, hospitality, high-end real estate, premium brands.
- *Cliché-trap:* **"serif + gold + black = luxury"** stereotype; thin-Didone-on-black default; slow motion that's just slow, not choreographed.
- *Signature:* **sensory storytelling** ("immersion conveys sensory craftsmanship"); art-directed full-bleed imagery/video, exquisite type detail (a real Didone paired well), **choreographed** slow motion, restraint as the signal, material/texture richness.
- *Pairs-with:* clip-wipe, scroll-scrub, cinematic scroll, grain, view transitions, slow parallax.
- *Currency:* evergreen; the **immersive-luxury** tier is hot.

## 10 · Technical / utilitarian (terminal / blueprint)
- *What:* honest-machine aesthetic — mono type, data density, grids, schematic/blueprint lines, terminal/CLI cues, engineered precision.
- *Fits:* dev tools, infra, data products, AI/ML, hardware — selling precision/trust.
- *Cliché-trap:* the **mono + grid + green-on-black terminal** cliché; **fake-technical decoration** (REV-A / FIG.01 / invented part numbers); data density that's actually unreadable.
- *Signature:* **real information design** — the structure encodes *true* data; mono with real hierarchy; schematic lines that *mean* something; restraint; one precise accent. Honest, not cosplay-technical.
- *Pairs-with:* scramble/counter, data viz, stateful hover, grid reveal.
- *Currency:* strong for tech/AI (precision/stability demand).

## 11 · Cinematic / scrollytelling *(experiential)*
- *What:* the page as a film — scroll-driven narrative scenes, pinned visuals + text steps, choreographed reveals, pacing, a story spine.
- *Fits:* product launches, brand stories, data stories, how-it-works — anything with a narrative.
- *Cliché-trap:* **scroll-jacked storytelling with no story** (the fatigue point); every section a "scene" with no spine; janky steps; forcing simple content into a heavy rig.
- *Signature:* **one defended narrative**, **scene logic like a curated film**, the **mechanic carries the message**, restraint, pacing that respects the reader, reduced-motion honored; few meaningful steps, not many. **The dual-view escape hatch:** a long immersive scroll earns its indulgence by shipping the SAME content ALSO as a scannable non-linear index (a "Full view / Overview" split), so the utilitarian visitor who just wants the facts is never trapped by the story — this is the actionable form of "the usability floor holds even at the top" (ref: learned-references → [[Relats]] Top Tier). **A defended DUALITY can organize the whole site more broadly:** Relats splits by *access* (immersion vs efficiency); [[Lando Norris]] splits by *identity* (on-track machine-self vs off-track human-self, which becomes the actual nav). One committed split carried through architecture, nav and hero is a repeatable top-tier spine.
- *Pairs-with:* scroll-scrub, pinned/scrollytelling, scroll-zoom, clip-wipe, view transitions, parallax.
- *Currency:* **matured into a craft.**

## 12 · 3D-immersive / spatial *(experiential — the apex)*
- *What:* real-time worlds — spatial navigation (walk/fly through), interactive 3D objects, camera-driven exploration; the apartment-walkthrough tier; **gaussian splatting** for photoreal spaces. **Real-time ≠ heavy WebGL 3D:** the lightweight sibling is **interactive vector (Rive state machines)** — recolorable, scroll/hover-reactive artboards at KB not MB — which reaches the same "alive, real-time, gives-you-agency" ceiling for logos/UI/mascots/objects without a Three.js pipeline. Reach for it first when the subject is a mark or a character, not a space.
- *Fits:* products that *are* spatial (architecture, hardware, cars, games), brand showpieces, craft-is-the-product portfolios — **and a PERSON / personal brand rendered as a living reactive SYSTEM** (an athlete, artist, founder: logo + UI + signature + a hero object all driven by one interactive runtime). **Only when content justifies real-time rendering** (heavy 3D especially — the vector sibling is cheap enough to spend more freely).
- *Cliché-trap:* a 3D blob with no purpose; heavy models that stall; spectacle with no concept; **no fallback** (broken on non-WebGPU / no reduced-motion path); 3D where a sharp photo converts better; real-time interactivity that's actually **on-rails** (scroll/hover only *triggers* canned states — never lets the user truly explore/spin/recolor).
- *Signature:* **the real-time IS the story** (drive-through portfolio; product-explore; a **living-identity system** where every mark reacts and recolors per section); intuitive control (guided camera / hotspots, or genuine hover/drag agency); **performance as craft** (LOD, compression, loading-as-experience); gaussian splatting for photoreal; graceful fallback. This is the concrete answer to passive video — real-time interactive lets the user EXPLORE, not watch (ref: learned-references → [[Lando Norris]] via Rive; the exceed-target it answers from [[Relats]]). Builder-assisted (heavy 3D) → needs-library (the vector sibling).
- *The CEILING of this movement — the whole page AS a real-time engine (the "one-canvas app"):* the maximal form isn't a 3D object *on* a page — it's the page *being* a single GPU application. The reference-ceiling is **Active Theory (Hydra engine)**: everything (text via MSDF, the entire UI, cursor, transitions, a multi-scene scroll journey) rendered in ONE `<canvas>` — a refractive-lens logo the reel plays *through*, a spine as the literal backbone the project cards branch off, cards that resolve to live video on hover and click into interactive scenes, navigation-as-experience (an AI assistant, QR mobile-multiplayer, a music player). Two load-bearing lessons come with the ceiling: **(a) reach for this only when the CRAFT itself is the product** — a real-time studio, a game, a spatial brand — so the spectacle is *earned* (medium = message), not [[Vanity Studio]]-style decoration; **(b) if you go full-canvas you OWE the web its contract back** — a real screen-reader/a11y tree + crawlable text (AT's `GLA11y`/`GLSEO`) + a low-power/reduced-motion path, or the flex is irresponsible. (ref: learned-references → [[Active Theory]].)
- *The REUSABLE mid-weight target — don't default to the ceiling.* The apex (Hydra-class engine) is the ceiling to *aim past on weight*, almost never the thing to ship. The fittest target for most experiential briefs sits **between the poles**: heavier than [[Vanity Studio]]'s pure-CSS carousel, far lighter than Active Theory's engine — **a single-`<canvas>`, single-bundle WebGL world that is fully meaning-locked.** Reference = **Samana Studio**: one canvas, one Vite bundle, Geist-only, a ~20.7k-px descent (≈½ AT's length) that still carries a real 3-D world (subsurface-scattering forms, volumetric god-rays, a Lenis-smoothed camera dive) — meaning-locked where Vanity was decoration, shippable where AT is a 30-second load. When a brief wants "immersive WebGL," this mid-weight world — not the one-canvas engine — is usually the honest answer. (ref: learned-references → [[Samana Studio]], the concrete occupant of the weight-axis middle.)
- *The metaphor can BE the pitch (spatial), and navigation can run on TWO axes.* When the subject sells an **intangible** (immersion, depth, calm, speed, precision), the strongest #12 move is to build the *literal spatial experience of that intangible* rather than assert it — the metaphor is the proof. Samana sells "immersion / felt-not-seen" and ships a **descent into the deep**: scroll = diving down, quantified by a **depth-meter HUD in real metres** (a real-unit readout reads as wayfinding *and* drama — better than a progress bar), and the manifesto beats are light-filled *presences* you pass. It also shows a repeatable IA: **two coupled navigation axes** — a vertical spine (scroll = deeper) crossed with an **in-scene focus axis** (click a 3-D-anchored waypoint → the camera dollies *closer* to a point on the object + reveals prose), giving a one-page site genuine explorable depth without more pages. This is the meaning-locked, lightweight cousin of the [[Relats]]/[[Lando Norris]] "defended organizing idea" — here the organizing idea is a *spatial metaphor of the pitch itself.* (ref: learned-references → [[Samana Studio]].)
- *Pairs-with:* interactive-3D-viewer, interactive-vector-state-machines (Rive), spatial-walkthrough / gaussian-splat, GPU particles, scroll-scrub, refractive/dispersion-glass material, refractive media cards, virtualized scroll journey.
- *Currency:* **WebGPU made the heavy tier more accessible;** the interactive-vector sibling makes the "alive" effect broadly shippable; the full one-canvas engine (Hydra-class) remains **builder-assisted, top-0.1%** — the ceiling to aim past on weight, not a floor to ship from. The apex tier.

## 13 · Generative / creative-coding *(experiential)*
- *What:* code-as-medium — generative art backdrops, particle/flow fields, shaders, audio/pointer-reactive systems, living algorithmic visuals.
- *Fits:* AI/tech/creative-coding brands, music, data-as-art, ambient living backdrops.
- *Cliché-trap:* the **connected-dots particle network** (instant generic tech tell); generative noise with no concept; GPU drain on mobile; a screensaver with no meaning.
- *Signature:* the system **means** something (maps to brand/data); restraint (one living element, not a screensaver); **reactive** (pointer/audio/scroll) so it feels alive; performance + fallback; WebGPU/TSL for the heavy tier (builder-assisted).
- *Pairs-with:* particles, GPU-compute particles, shaders, mesh gradient, custom cursor.
- *Currency:* **WebGPU-enabled;** strong for AI/creative brands.
